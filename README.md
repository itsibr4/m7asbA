# نظام المحاسبة المتكامل - الإصدار 2.1

برنامج سطح مكتب احترافي لإدارة المبيعات والمخزون والأرباح، مبني بلغة Python مع واجهة رسومية عصرية.

## ✨ مميزات الإصدار 2.1

- **📊 لوحة التحكم (Dashboard)**: إحصائيات شاملة ومباشرة
- **💱 الدينار العراقي (IQD)**: العملة الافتراضية مع دعم العملات الأخرى
- **⚙️ إعدادات العملة**: تغيير العملة والمنازل العشرية من الإعدادات
- **📄 ترقيم الصفحات**: دعم +1000 منتج مع تقسيم الصفحات
- **↕️↔️ Scrollbars**: أفقية وعمودية للجداول الكبيرة
- **تصميم احترافي**: واجهة مستخدم عصرية بألوان محاسبية أنيقة

## 🚀 طريقة التشغيل

### الطريقة السهلة (Windows)
1. فك ضغط الملف
2. اضغط مرتين على `تشغيل_البرنامج.bat`
3. يثبت المكتبات ويشغل البرنامج تلقائياً!

### الطريقة اليدوية
```bash
pip install -r requirements.txt
python main.py
```

### بناء ملف EXE
```bash
pip install pyinstaller
python build_exe.py
```

## 📄 ترقيم الصفحات

للتعامل مع +1000 منتج:
- اختر عدد العناصر لكل صفحة: 25, 50, 100, 200, 500
- أزرار التنقل: السابق / التالي
- بحث فوري يعمل على كل المنتجات
